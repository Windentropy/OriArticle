# 大结局篇 
经历了分离以后，人间体的意识也变得更加清晰明显，我也预料到了这个孩子如此强大的内心能力，毕竟，如果不是当初他那强烈的呼唤，我也不会来到这个时间段短地球并与他融为一体。如今，这个身体的主人有了更强烈的欲望，我也当然要满足他了。  
“奥里，想环游地球嘛？甚至环游这个宇宙。”  
“怎么突然提到这个？”  
“是人间体的想法，他想和你漫游整个世界，想和你旅行，借着我这个媒介每天过着一成不变的生活，只是早安晚安要抱抱或多或少都也很无聊，他想尝试一点新奇的东西。”  
“可以啊，我无所谓的，只要你开心，他开心，我也就开心了。”  
“行，那我准备一下，这次就以人类的方式旅游吧，到时候还得给你办张身份证啥的，还有签证啊，护照啊，户口簿啊，这些就落到我妈那吧，办起来倒简单，一天他们基本就能整出来，你那边有啥需要准备的嘛？”  
“没有啥啊，就跟上次和你去上海一样嘛。”  
“行，那我明天就去办事儿了。”  
我和奥里在餐桌上一边吃饭一边交谈，饭后我列好了准备清单，今天就准备基本的生活用品和旅游攻略，明天就去人类世界办理Ori的相关证明了。或许是人间体知晓了这个的原因，表现的很活跃，我的能量也受到了影响，似乎变得更加强大了。或许这就是同化带来的影响吧。  
最终，旅行的日期选在了3月11号，我和奥里早上7点把行李打包好，我背包，奥里提箱子，我们高高兴兴地出门了。临行前，我以网络为媒介，将认知魔法施加给全世界，这样，所有人都会以为Ori只是个普通的人类，不会因为他的出现而感到好奇了。这几个月里，我们领略到了埃及的神秘传说，亚马逊的狂野自然，旧金山的现金科技，夏威夷的燥热火山，北欧的恬静生活，极地的冰点美景，西藏的壮阔高原一路上，我们住过酒店，去过朋友家做客，住过房车，在野外野营。Ori在地球环境的熏陶共鸣之下，接受了太阳之光的洗礼，森林之力的熏染，冷热温差的冲击，所表现出来的精灵之力也愈发强大，他的能力从最初的精灵之光开始向自然元素发展演变了，或许以后真的会变成五彩的Ori吧。  
环游完世界主要大都市以及自然风光后，我和Ori回到了中国，在北京某个酒店暂住。某天夜里，我和Ori在酒店阳台上享受着北京晴朗的夜空（空气污染已经治理完了），我问Ori:“亲爱的，你知道龙吗？那是中国古代一种神话生物，在他们的眼里是不存在的。龙保佑着一国平安，祝福一国风调雨顺，平安祥和。在他们眼里，龙便是至高无上的存在。”  
“这样吗？那龙到底长什么样子呢？”  
“你想看吗？等一下。”  
这是我为亲爱的准备的一个惊喜。
 北京的夜晚，充满着现代大都市的韵味，五彩斑斓的灯光覆盖掉黑暗的区域，滴滴作响的汽车声、叽喳的人声和天上呜呜飞过的飞机声杂糅在一起，与曾经的老旧古城有了明显的区别，上次来北京还是几百年前吧，人类经过几百年的耕耘，让这片土地从最初的沉寂升华到了如今的繁盛。  
“所以到底是啥惊喜呢？”  
我站在酒店顶楼的天台上，静静的望着这座不可能睡去的大城市，深呼吸了一口，而后轻轻飘起，在空中上升同时旋转几周，我的身形逐渐拉长，头上长出两根角和触须，脸逐渐拉长，身体长出鳞片，四肢变成了五爪，在空中漂浮。  
“这便是人们最敬仰的龙的样子。”我说罢便飞了下来，来到ori身边，微微低头，示意Ori坐上来。最后我们俩在天上看完了北京全貌，飞向宇宙，俯瞰整个地球。在这一切都结束以后，回到了尼博尔。  
在尼博尔的那个夜里，我对Ori说:“我会一直陪着你，保护你的，只要你一直在......只要你一直在........只要你......”  
(闹钟音乐)  
“醒了啊？快去吃早饭，都睡了多久了。”  
“我睡了很久吗？”  
“是啊，都下午了，你昨晚干了啥啊？”  
“我......没干啥......”  
“只是，有点不舍而已。” “什么不舍哦，你又做了什么梦嘛。” “害，没啥，有啥吃的吗？我要饿死了。” “等晚上吃完饭吧，我买了些零食，你先吃点吧。” “好吧，谢谢妈。” (完)
